BENDING TREE SYSTEM - README
--------------------------------------------------------------
OVERVIEW
--------------------------------------------------------------
The Bending Tree System is a subgraph to add to your own Shader Graph.  
It allows trees and foliage to bend dynamically in response to the  
Wind Manager script.

This package is compatible with Grass Physics and Wind.  
It allows your mesh to bend based on wind direction and intensity.

One shader is designed for trunks and a second shader is made for leaves  
(with a variant for alpha rendering).

This system is optimized for stylized visuals, mobile devices,  
and VR projects.
--------------------------------------------------------------
FEATURES
--------------------------------------------------------------
* SubShader Graph-based wind bending (URP compatible)
* Ready-to-use shaders for trunks and leaves (opaque or alpha render)
* Wind Manager with strength, size, speed, and direction settings
* Customizable material to set up the system
* Ready-to-use tree prefabs
* Included demo scene showing setup and usage
* Lightweight and optimized for performance
--------------------------------------------------------------
HOW TO USE
--------------------------------------------------------------
1. IMPORT PACKAGE  
   Import the Bending Tree System into your Unity project (URP required).

2. ADD WIND MANAGER  
   Drag the 'WindController' prefab into your scene.

3. DRAG AND DROP TREE PREFABS  
   Use the included prefabs or apply the shaders to your own models.

4. CONFIGURE WIND  
   Select the WindController object and adjust settings in the Inspector:  
   - Wind direction  
   - Wind strength  
   - Wind speed
